# Mapa + QR Composer (Streamlit)

Aplicación que compone automáticamente una imagen con:
- Nombre (arriba a la izquierda)
- QR (izquierda, debajo del nombre)
- Mapa (derecha)

Cómo ejecutar localmente (opcional):
1. python3 -m venv venv
2. source venv/bin/activate
3. pip install -r requirements.txt
4. streamlit run streamlit_app.py
